
**Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

**Start the development server**

   ```bash
   expo start
   ```

    * Scan the QR code with the Expo Go app on your phone **or**
    * Run on a simulator:

        * Press `i` for iOS
        * Press `a` for Android
